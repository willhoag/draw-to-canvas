# Security Policy

## Project status

This project is no longer maintained.

Known security vulnerabilities may exist in this repository and its dependencies. Security reports are not actively triaged.

Please do not use this project in production. Forks or users of this code should audit dependencies and replace vulnerable packages before use.

## Supported versions

No versions are currently supported.

## Reporting vulnerabilities

Vulnerability reports may not receive a response because this project has been retired.
